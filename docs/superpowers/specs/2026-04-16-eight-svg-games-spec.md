# 8 New SVG Mini-Games Spec — Wave 3

## 공통 규약

- 모든 게임: `games/<폴더>/` 하위 index.html, style.css, game.js, game.json
- `../../shared/style.css`, `../../shared/engine.js` 로드
- **이모지 사용 금지** — 모든 아이콘/캐릭터/아이템을 인라인 SVG로 제작
- 영역 분할 + 동시 플레이 (2~4인)
- 탭만 조작, 30초~2분
- 사운드: Web Audio API, 기본 음소거
- setTimeout 사용 (requestAnimationFrame 금지 — 비활성 탭 이슈)

## SVG 가이드라인

- 모든 그래픽: 인라인 `<svg>` 태그 사용 (외부 파일 X)
- viewBox 기반 — 크기는 CSS로 조절
- 색상: 디자인 토큰 기반 (밝고 친근한 톤)
- 스타일: 둥근 형태, 두꺼운 선(2-3px), 밝은 색 채우기
- 게임 카드 아이콘: 각 game.json의 icon 필드에 SVG를 넣을 수 없으므로 이모지 유지 (런처 카드 썸네일만)
- 게임 내부 모든 그래픽 요소: SVG

## 1. 풍선 터트리기 (Balloon Pop)

**폴더:** `balloon-pop`
**인원:** 2~4인
**시간:** 30초

**룰:**
- 화면을 플레이어 수만큼 영역 분할 (2인: 좌/우, 4인: 4분할)
- 각 영역에서 풍선 SVG가 아래에서 위로 떠오름
- 풍선을 터치하면 터짐 (+1점). 위로 날아가면 놓침
- 30초 후 가장 많이 터트린 사람 승리
- 풍선 색상: 랜덤 (빨/파/초/노/주/보 6색)
- 시간 경과에 따라 풍선 속도 증가

**SVG 풍선:**
- 타원형 몸체 + 아래 삼각 꼭지 + 실 (곡선)
- 하이라이트(빛 반사) 포함
- 터질 때: scale 커지면서 opacity 0 애니메이션

**game.json:**
```json
{ "name": "풍선 터트리기", "description": "떠오르는 풍선을 빠르게 터치!", "icon": "🎈", "color": "#E91E63", "grades": [1,2,3,4,5,6], "playTime": "30초" }
```

## 2. 색깔 터치 (Color Touch / Stroop)

**폴더:** `color-touch`
**인원:** 2~4인
**시간:** 10라운드, ~1분

**룰:**
- 중앙에 색 이름 텍스트 표시 (예: "빨강")
- 단, 텍스트의 글자색이 의미와 다를 수 있음 (스트룹 효과: "빨강"이 파란색 글씨로 표시)
- 각 영역에 4개 색 버튼 (빨/파/초/노) — SVG 원형
- **텍스트의 의미(뜻)에 해당하는 색 버튼**을 먼저 터치한 사람이 +1점
- 오답: 해당 라운드 실격
- 10라운드 후 총점 비교

**SVG 색 버튼:**
- 큰 원형, 각 색상으로 채움
- 터치 시 scale 애니메이션

**game.json:**
```json
{ "name": "색깔 터치", "description": "글자의 뜻에 맞는 색을 터치!", "icon": "🟡", "color": "#9C27B0", "grades": [1,2,3,4,5,6], "playTime": "1분" }
```

## 3. 폭탄 피하기 (Bomb Dodge)

**폴더:** `bomb-dodge`
**인원:** 2~4인
**시간:** 30초

**룰:**
- 각 영역에 아이템이 랜덤 위치에 등장 (1~2초 후 사라짐)
- 별(SVG) 터치: +1점
- 폭탄(SVG) 터치: -1점
- 아무것도 안 터치: 사라짐 (놓침)
- 30초 후 최고 점수 승리
- 별:폭탄 비율 = 약 7:3

**SVG 별:** 5각 별, 노란색, 반짝이는 하이라이트
**SVG 폭탄:** 검은 원형 + 도화선 + 불꽃

**game.json:**
```json
{ "name": "폭탄 피하기", "description": "별은 터치! 폭탄은 피해!", "icon": "💣", "color": "#F44336", "grades": [1,2,3,4,5,6], "playTime": "30초" }
```

## 4. 모양 맞추기 (Shape Match)

**폴더:** `shape-match`
**인원:** 2~4인
**시간:** 10라운드, ~1분

**룰:**
- 중앙에 목표 아이템 SVG 표시 (동물, 과일, 사물 등 20종 풀)
- 각 영역에 6개 보기 SVG 표시 (정답 1개 + 오답 5개)
- 정답을 먼저 터치한 사람이 +1점
- 오답 터치: 해당 라운드 실격
- 10라운드

**SVG 아이템 풀 (20종):**
- 동물: 고양이, 강아지, 토끼, 물고기, 새
- 과일: 사과, 바나나, 포도, 딸기, 수박
- 사물: 별, 하트, 달, 구름, 번개, 나무, 꽃, 자동차, 로켓, 집

**game.json:**
```json
{ "name": "모양 맞추기", "description": "같은 그림을 가장 먼저 찾아라!", "icon": "🔷", "color": "#00BCD4", "grades": [1,2,3,4,5,6], "playTime": "1분" }
```

## 5. 크기 비교 (Bigger or Smaller)

**폴더:** `size-compare`
**인원:** 2~4인
**시간:** 15라운드, ~1분

**룰:**
- 중앙에 두 숫자 SVG로 크게 표시 (예: 37 vs 82)
- 각 영역에 "왼쪽" / "오른쪽" 버튼 2개 (SVG 화살표)
- 큰 숫자 쪽을 먼저 터치한 사람 +1점
- 오답: 해당 라운드 실격
- 15라운드 후 총점 비교
- 난이도: 쉬움(1자리 vs 1자리), 보통(2자리 vs 2자리)

**SVG 숫자:** 큰 볼드 텍스트를 SVG로 렌더링 (배경 원형 안에)
**SVG 화살표:** ◀ ▶ 형태의 큰 화살표 버튼

**game.json:**
```json
{ "name": "크기 비교", "description": "더 큰 숫자를 먼저 터치!", "icon": "📐", "color": "#FF9800", "grades": [1,2,3,4,5,6], "playTime": "1분" }
```

## 6. 빠른 분류 (Quick Sort)

**폴더:** `quick-sort`
**인원:** 2~4인
**시간:** 30초

**룰:**
- 상단에 카테고리 표시 (예: "과일만 터치!")
- 각 영역에 SVG 아이템 8개 등장 (카테고리에 해당하는 것 4개 + 아닌 것 4개)
- 해당하는 것 터치: +1점
- 해당하지 않는 것 터치: -1점
- 맞게 터치한 아이템은 사라지고 새 아이템 등장
- 30초 후 점수 비교

**카테고리:**
- 과일 (사과,바나나,포도,딸기,수박,체리,복숭아,키위)
- 동물 (고양이,강아지,토끼,물고기,새,곰,펭귄,여우)
- 탈것 (자동차,버스,비행기,배,자전거,로켓,기차,헬리콥터)

**SVG 아이템:** 모양 맞추기의 SVG 풀을 확장하여 공유

**game.json:**
```json
{ "name": "빠른 분류", "description": "맞는 것만 골라서 터치!", "icon": "🗂️", "color": "#4CAF50", "grades": [1,2,3,4,5,6], "playTime": "30초" }
```

## 7. 순간 기억 (Memory Flash)

**폴더:** `memory-flash`
**인원:** 2~4인
**시간:** 10라운드, ~2분

**룰:**
- 각 영역에 3x3 격자 (9칸)
- 라운드 시작: 3~5개 칸이 1.5초간 노란색으로 반짝인 후 사라짐
- 플레이어가 기억해서 같은 칸을 터치
- 맞춘 칸 수만큼 +점 (정확히 모든 칸 맞추면 보너스 +2)
- 틀린 칸 터치: -1점
- 제한시간: 라운드당 5초
- 모든 플레이어가 동시에 각자 영역에서 터치 (동시 플레이)
- 10라운드, 라운드마다 반짝이는 칸 수 증가 (라운드1: 3개, 라운드5: 4개, 라운드8: 5개)

**SVG:** 격자 칸은 SVG rect, 반짝임은 fill 색상 변경 + glow 효과

**game.json:**
```json
{ "name": "순간 기억", "description": "반짝인 칸을 기억하고 터치!", "icon": "🧠", "color": "#673AB7", "grades": [1,2,3,4,5,6], "playTime": "2분" }
```

## 8. 짝홀 대결 (Even or Odd)

**폴더:** `even-odd`
**인원:** 2~4인
**시간:** 15라운드, ~1분

**룰:**
- 중앙에 숫자 SVG로 크게 표시 (1~99)
- 각 영역에 "짝수" / "홀수" 버튼 2개 (SVG)
- 정답을 먼저 터치한 사람 +1점
- 오답: 해당 라운드 실격
- 15라운드 후 총점 비교

**SVG 버튼:** 둥근 사각형 안에 "짝" / "홀" 텍스트, 짝수=파랑, 홀수=빨강

**game.json:**
```json
{ "name": "짝홀 대결", "description": "짝수? 홀수? 빠르게 판단!", "icon": "🔢", "color": "#3F51B5", "grades": [1,2,3,4,5,6], "playTime": "1분" }
```
