# 10 Mini-Games Spec — Killing Time Collection

공통 규약:
- 모든 게임은 `games/<폴더명>/` 하위에 `index.html`, `style.css`, `game.js`, `game.json` 구성
- `../../shared/style.css`와 `../../shared/engine.js`를 로드
- 화면 흐름: 설명(PLAY 버튼) → 게임 플레이 → 결과(다시하기/홈으로)
- 2인 이상 게임: 설명 화면에서 인원 선택 후 PLAY
- 사운드: `createSoundManager`로 Web Audio API 효과음 (기본 음소거)
- 터치/클릭 통합: `onTap` 사용
- 점수: 세션만, 저장 없음

---

## 1. 신호 반응 (Reaction Race)

**폴더:** `reaction-race`
**인원:** 2~4인
**게임 시간:** 5라운드, 약 1분

**룰:**
- 화면에 플레이어별 큰 컬러 버튼 표시 (2인: 좌/우, 4인: 4분할)
- "기다려..." 텍스트 표시 후 1~4초 랜덤 대기
- 화면 전체가 초록색으로 바뀌면 자기 버튼을 먼저 터치
- 너무 일찍 누르면 (초록 전) 해당 라운드 실격
- 5라운드 중 가장 많이 이긴 사람 승리

**화면 구성:**
- 설명 화면: 인원 선택 (2/3/4) + PLAY
- 게임: 플레이어별 컬러 영역 (파랑/빨강/초록/주황), 중앙에 상태 텍스트
- 결과: 라운드별 결과표 + 최종 승자

**조작:** 자기 색 영역 터치
**점수:** 라운드 승리 횟수

## 2. 숫자 순서대로 (Number Tap Race)

**폴더:** `number-tap`
**인원:** 2~4인
**게임 시간:** 1~2분

**룰:**
- 각 플레이어 영역에 1~15 숫자가 랜덤 위치에 흩어짐
- 1부터 순서대로 터치해야 함. 틀리면 반응 없음
- 먼저 15까지 완료한 사람 승리
- 난이도: 저학년 1~10, 고학년 1~20

**화면 구성:**
- 설명 화면: 인원 선택 + 난이도 선택 (쉬움/보통) + PLAY
- 게임: 화면 분할, 각 영역에 원형 숫자 버튼 흩어짐. 현재 눌러야 할 숫자 상단 표시
- 결과: 완료 시간 또는 순위

**조작:** 숫자 터치
**점수:** 완료 순서 (1등/2등...)

## 3. 페인트 파이트 (Paint Fight)

**폴더:** `paint-fight`
**인원:** 2~4인
**게임 시간:** 30초

**룰:**
- 화면 전체에 8x8 또는 6x6 격자
- 각 플레이어에게 고유 색상 배정
- 빈 칸 또는 상대 칸을 터치하면 자기 색으로 칠해짐
- 30초 후 칸 수가 가장 많은 사람 승리

**화면 구성:**
- 설명 화면: 인원 선택 + PLAY
- 게임: 격자 그리드, 상단에 플레이어별 점수(칸 수) 실시간 표시. 타이머 바
- 결과: 색칠된 격자 + 최종 칸 수 비교

**조작:** 격자 칸 터치
**점수:** 칠한 칸 수
**참고:** 멀티터치 — 동시에 여러 명이 터치 가능해야 함 (각 touch의 위치로 가장 가까운 칸 칠하기)

## 4. 틱택토 (Tic Tac Toe)

**폴더:** `tic-tac-toe`
**인원:** 2인
**게임 시간:** 1~2분

**룰:**
- 3x3 격자에 번갈아 O/X 놓기
- 가로/세로/대각선 3개 연속 먼저 만들면 승리
- 모든 칸이 차면 무승부

**화면 구성:**
- 설명 화면: PLAY 버튼 (인원 선택 불필요, 항상 2인)
- 게임: 큰 3x3 격자 중앙 배치. 상단에 "O의 차례" / "X의 차례" 표시. O는 파랑, X는 빨강
- 결과: 승자 표시 또는 무승부 + 다시하기/홈으로

**조작:** 빈 칸 터치
**점수:** 승/패/무승부

## 5. 사목 (Connect 4)

**폴더:** `connect-four`
**인원:** 2인
**게임 시간:** 2~3분

**룰:**
- 7열 x 6행 세로 보드
- 번갈아 열 선택 → 해당 열의 가장 아래 빈 칸에 돌 떨어짐
- 가로/세로/대각선 4개 연속 먼저 만들면 승리

**화면 구성:**
- 설명 화면: PLAY 버튼
- 게임: 세로 보드 (파란 배경), 상단에 열 선택 화살표 또는 열 터치. 플레이어1 빨강, 플레이어2 노랑. 현재 차례 표시
- 결과: 승자 + 이긴 줄 하이라이트

**조작:** 열 상단 터치 → 돌 떨어지는 애니메이션
**점수:** 승/패

## 6. 도트 앤 박스 (Dots and Boxes)

**폴더:** `dots-and-boxes`
**인원:** 2~4인
**게임 시간:** 2~3분

**룰:**
- 5x5 점 격자 (4x4 칸)
- 번갈아 두 점 사이에 선을 그음
- 사각형의 네 번째 변을 완성하면 자기 색으로 칠해지고 한 번 더
- 모든 칸이 채워지면 가장 많은 칸을 가진 사람 승리

**화면 구성:**
- 설명 화면: 인원 선택 (2/3/4) + PLAY
- 게임: 점 격자, 선은 터치로 그음 (두 인접 점 사이 공간 터치). 플레이어별 색상. 현재 차례 표시. 각 플레이어 점수
- 결과: 완성된 보드 + 점수

**조작:** 두 점 사이 공간 터치
**점수:** 완성한 사각형 수

## 7. 양궁 (Archery)

**폴더:** `archery`
**인원:** 2~4인
**게임 시간:** 1~2분 (각 3발씩)

**룰:**
- 과녁 (동심원 5단계: 10, 8, 6, 4, 2점)
- 터치 후 드래그로 조준 방향/세기 설정 → 손 떼면 발사
- 바람 요소: 상단에 풍향/풍속 표시, 발사마다 변경
- 각 플레이어 3발씩, 총점 비교

**화면 구성:**
- 설명 화면: 인원 선택 + PLAY
- 게임: 과녁 중앙 배치, 하단에 활/발사 영역. 상단에 바람 표시 + 현재 플레이어 + 남은 화살. 점수판
- 결과: 플레이어별 총점 비교

**조작:** 드래그 (방향+세기) → 릴리즈로 발사
**점수:** 과녁 점수 합산

---

## 8. 배틀쉽 (Sea Battle)

**폴더:** `battleship`
**인원:** 2인
**게임 시간:** 3~5분

**룰:**
- 각 플레이어 8x8 바다 격자
- 배치 단계: 배 3척 (3칸, 2칸, 2칸) 자기 격자에 배치 (터치로 위치 선택, 버튼으로 회전)
- 공격 단계: 번갈아 상대 격자의 칸 터치 → 히트(빨강)/미스(회색) 표시
- 상대 배 3척 모두 침몰시키면 승리

**화면 구성:**
- 설명 화면: PLAY 버튼
- 배치 화면: "플레이어1 배 배치" → 격자 + 배 + 회전 버튼 + 완료 버튼. 플레이어2는 화면 돌려서 배치
- 게임: 상단에 상대 격자 (공격용), 하단에 내 격자 (방어 상황). 현재 차례 표시
- 결과: 승자 + 양쪽 격자 공개

**조작:** 격자 칸 터치
**점수:** 승/패
**참고:** 턴 전환 시 "화면을 상대에게 넘기세요" 메시지 + 터치하면 다음

## 9. 농구 슛 (Basketball Hoops)

**폴더:** `basketball`
**인원:** 2~4인
**게임 시간:** 1~2분 (각 5발)

**룰:**
- 화면 하단에 농구공, 상단에 골대
- 공을 위로 스와이프(드래그)하여 던짐 — 드래그 방향과 속도가 궤적 결정
- 골인 시 2점, 림에 맞으면 1점, 미스 0점
- 각 플레이어 5발씩, 총점 비교

**화면 구성:**
- 설명 화면: 인원 선택 + PLAY
- 게임: 배경 체육관 색상, 골대(상단), 공(하단). 궤적 포물선 애니메이션. 현재 플레이어 + 남은 공 + 점수판
- 결과: 플레이어별 총점 비교

**조작:** 공을 위 방향으로 드래그+릴리즈 (스와이프)
**점수:** 골인 점수 합산

---

## 10. 파리 잡기 (Catch the Fly)

**폴더:** `catch-the-fly`
**인원:** 2~4인
**게임 시간:** 30초

**룰:**
- 화면을 플레이어 수만큼 분할 (2인: 좌/우, 4인: 4분할)
- 각 영역에 파리가 랜덤 위치에 등장 → 1~2초 후 사라짐
- 파리를 터치하면 +1점
- 30초 후 가장 많이 잡은 사람 승리
- 두더지 잡기와 차이: 영역 대결, 파리가 구멍 없이 자유롭게 날아다님

**화면 구성:**
- 설명 화면: 인원 선택 + PLAY
- 게임: 분할된 영역, 각 영역 다른 배경색. 파리 SVG가 랜덤 위치에 등장. 상단에 플레이어별 점수
- 결과: 점수 비교 + 승자

**조작:** 파리 터치
**점수:** 잡은 파리 수
**파리 비주얼:** SVG 파리 (날개 퍼덕 CSS 애니메이션)
