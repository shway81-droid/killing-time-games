# 5 New Mini-Games Spec — Killing Time Collection (Wave 2)

공통 규약 (기존과 동일):
- `games/<폴더명>/` 하위에 index.html, style.css, game.js, game.json
- `../../shared/style.css`와 `../../shared/engine.js` 로드
- 화면 흐름: 설명(PLAY) → 게임 → 결과(다시하기/홈으로)
- 2인 이상: 설명 화면에서 인원 선택
- 사운드: Web Audio API, 기본 음소거
- 조작: 탭만 (드래그/스와이프 없음)
- 점수: 세션만

---

## 1. 카드 뒤집기 (Memory Match)

**폴더:** `memory-match`
**인원:** 2~4인
**시간:** 1~2분

**룰:**
- 4x4 카드 그리드 (8쌍 = 16장). 뒷면은 모두 동일 색상
- 턴제: 현재 플레이어가 카드 2장을 탭해서 뒤집음
- 같은 이모지 쌍이면: 해당 플레이어 점수 +1, 카드 제거(또는 색 표시), **한 번 더**
- 다른 쌍이면: 1초 후 다시 뒤집히고, 다음 플레이어 턴
- 모든 쌍 매칭 시 가장 많은 쌍을 가진 사람 승리

**이모지 풀:** 🐶🐱🐸🦊🐼🐨🦁🐯🐷🐮🐵🐔🦄🐙🦋🐝 (이 중 8개 랜덤 선택)

**화면:**
- 설명: 인원 선택(2/3/4) + PLAY
- 게임: 4x4 카드 그리드 중앙 배치. 상단에 현재 턴 표시 + 플레이어별 점수. 카드 뒤집기 애니메이션 (CSS 3D transform)
- 결과: 최종 점수 비교 + 승자

**game.json:**
```json
{ "name": "카드 뒤집기", "description": "같은 그림 카드 짝을 찾으세요!", "icon": "🃏", "color": "#AB47BC", "grades": [1,2,3,4,5,6], "playTime": "1~2분" }
```

## 2. 빠른 계산 (Quick Math)

**폴더:** `quick-math`
**인원:** 2~4인
**시간:** 1~2분

**룰:**
- 화면 중앙에 수식 표시 (예: "7 + 5 = ?")
- 4개 보기 버튼 표시 (정답 1개 + 오답 3개)
- 플레이어 영역 분할 (2인: 좌/우, 4인: 4분할). 각 영역에 동일한 4개 보기 표시
- 정답을 먼저 터치한 플레이어가 +1점
- 오답 터치 시: 해당 라운드 해당 플레이어 실격 (다른 플레이어가 맞출 기회)
- 10문제 후 총점 비교

**난이도:**
- 쉬움(저학년): 한 자리 덧셈/뺄셈 (1+3, 9-4)
- 보통(고학년): 두 자리 연산 + 곱셈 (15+28, 7×6)

**화면:**
- 설명: 인원 선택 + 난이도 선택(쉬움/보통) + PLAY
- 게임: 상단 중앙에 수식. 플레이어 영역별로 4개 보기 버튼. 문제 번호(1/10) + 점수판
- 결과: 총점 비교 + 승자

**game.json:**
```json
{ "name": "빠른 계산", "description": "정답을 가장 먼저 터치하세요!", "icon": "🔢", "color": "#26A69A", "grades": [1,2,3,4,5,6], "playTime": "1~2분" }
```

참고: 숫자 순서대로 게임이 이미 🔢 아이콘을 사용하므로, 빠른 계산은 **🧮** 아이콘으로 변경.

## 3. 색깔 따라하기 (Simon Says)

**폴더:** `simon-says`
**인원:** 2~4인
**시간:** 2~3분

**룰:**
- 4개 색 버튼 (빨/파/초/노) 화면 중앙에 크게 배치
- 라운드 시작: 시스템이 색 순서를 보여줌 (버튼이 순서대로 번쩍임)
- 라운드 1 = 1색, 라운드 2 = 2색, ... (누적 추가)
- 각 플레이어가 턴제로 같은 순서를 탭으로 입력
- 틀리면 탈락. 마지막까지 남은 사람 승리
- 모든 플레이어가 맞추면 다음 라운드 (색 1개 추가)
- 최대 15라운드

**화면:**
- 설명: 인원 선택(2/3/4) + PLAY
- 게임: 4색 버튼 (2x2 큰 사각형). 상단에 "보는 중..." / "P1의 차례" 표시. 탈락한 플레이어는 회색 표시
- 결과: 최종 탈락 순서 + 승자

**game.json:**
```json
{ "name": "색깔 따라하기", "description": "색 순서를 기억하고 따라하세요!", "icon": "🌈", "color": "#5C6BC0", "grades": [1,2,3,4,5,6], "playTime": "2~3분" }
```

## 4. 같은 그림 찾기 (Spot the Match)

**폴더:** `spot-the-match`
**인원:** 2~4인
**시간:** 1~2분

**룰:**
- 화면에 이모지 9개 표시 (3x3 그리드). 그 중 정확히 2개가 같음
- 같은 2개를 연속으로 탭한 사람이 득점 (+1)
- 틀리면 (다른 것 탭) 반응 없음, 다시 시도
- 맞추면 새로운 라운드 (이모지 셔플)
- 10라운드 후 총점 비교
- 멀티플레이어: 영역 분할 없이 **공용 화면**에서 경쟁. 먼저 2개를 맞춘 사람이 득점

**이모지 풀:** 다양한 동물/음식/사물 이모지 30종+ (매 라운드 8종 + 1종 중복 = 9개)

**화면:**
- 설명: 인원 선택 + PLAY
- 게임: 3x3 이모지 그리드 중앙. 상단에 라운드(1/10) + 플레이어별 점수. 누가 먼저 찾았는지 표시
- 결과: 총점 비교 + 승자

**참고:** 멀티플레이어에서 "누가 탭했는지" 구별이 어려우므로, 턴제로 변경 가능. 또는 단순히 "가장 먼저 두 번째 카드를 탭한 시점의 현재 턴 플레이어"로 처리. **간단한 접근: 턴제 — 현재 플레이어만 탭 가능, 찾으면 한 번 더, 못 찾으면(3초 타임아웃 또는 오답 2회) 다음 턴.**

**game.json:**
```json
{ "name": "같은 그림 찾기", "description": "같은 이모지 2개를 빨리 찾으세요!", "icon": "🔍", "color": "#FDD835", "grades": [1,2,3,4,5,6], "playTime": "1~2분" }
```

## 5. OX 퀴즈 (OX Quiz)

**폴더:** `ox-quiz`
**인원:** 2~4인
**시간:** 1~2분

**룰:**
- 화면에 문장 표시 (예: "지구는 태양 주위를 돈다")
- 플레이어 영역 분할, 각 영역에 O/X 버튼
- 정답을 먼저 터치한 플레이어가 +1점
- 오답 터치 시: 해당 라운드 실격
- 10문제 후 총점 비교

**문제 데이터: 300문제**
- game.js 내 배열로 하드코딩
- 카테고리: 과학(60), 수학(40), 국어/한국어(40), 사회/역사(50), 상식(50), 자연/동물(30), 지리(30)
- 형식: `{ q: "문장", a: true/false }`
- 매 게임마다 300개 중 10개 랜덤 선택
- 한국 초등학생 수준 (1~6학년)

**화면:**
- 설명: 인원 선택(2/3/4) + PLAY
- 게임: 상단에 문제 텍스트 (큰 글씨). 플레이어 영역별 O/X 버튼 (O=파랑, X=빨강). 문제 번호 + 점수판
- 결과: 총점 비교 + 승자

**game.json:**
```json
{ "name": "OX 퀴즈", "description": "O일까 X일까? 빠르게 맞춰보세요!", "icon": "⭕", "color": "#EC407A", "grades": [1,2,3,4,5,6], "playTime": "1~2분" }
```

참고: 틱택토가 ⭕ 아이콘을 사용하므로 OX 퀴즈는 **❓** 아이콘으로 변경.
